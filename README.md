# Layout de Teclado en Español (Chile) Latinoamérica para macOS

## Acerca de 
Este es un respaldo de una entrada antigua de mi [blog](https://web.archive.org/web/20220117063011/https://brede.cl/2013/02/28/layout-teclado-espanol-chile-latinoamerica-para-os-x/)

Hace unos 15 años tuve un notebook Dell que pedó con el [teclado Español](https://www.goodtyping.com/teclatLAT.htm), con el cual me era bastante fácil escribir código ya que no debía presionar combinaciones raras de teclas para obtener caractéres de uso común como los `{ } [ ] ? < >` que uso constantemente cuando programo en PHP.

### Carácteres de uso común
- Las llaves `{` y `}` se ejecutan directamente sólo presionando las teclas del lado derecho de la `ñ`.
- Los paréntesis cuadrados `[` y `]` sólo requieres presionar el shift en las mismas teclas que `{` y `}`.
- Para los *Backticks* sólo presionas la tecla del acento y el espacio (al lado de la `p`)

Ahora lo importante:
# Cómo instalarlo

1. Descarga el archivo `Latinoamericano.keylayout` desde el repo
2. Desde el finder, anda a la carpeta `/Users/{tu usuario}/Library/Keyboard Layouts/`
3. Pega el archivo en dicha carpeta
4. Dependiendo de tu idioma de macOS
    1. Si es Español, anda a `Ajustes del Sistema` > `Teclados` > `Entrada de Texto` y Selecciona `Chile Latinoamericano`
    2. Si es Inglés, anda a `System Settings` > `Keyboard` > `Text Input` y Selecciona `Chile Latinoamericano`
5. Te recomiendo desactivar otros layouts y dejar sólo el `Chile Latinoamericano`, o en su defecto mostrar el ícono en la barra superior para cambiarlo si es que llegase a fallar (imagen 3)
6. Listo, ya puedes usarlo.
